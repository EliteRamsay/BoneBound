BoneBound - Game
================

HOW TO PLAY:
- Arrow Keys or WASD to move
- F for Fullscreen
- Backspace to return to menu
- ESC to quit

SYSTEM REQUIREMENTS:
- Windows 7/8/10/11
- OpenGL 3.3 or higher support

INSTRUCTIONS:
1. Extract all files to a folder
2. Double-click BoneBound.exe to play
3. No installation needed!

TROUBLESHOOTING:
- If game doesn't start, install Visual C++ Redistributable:
  https://aka.ms/vs/17/release/vc_redist.x64.exe

- For OpenGL errors, update your graphics drivers

CONTROLS:
- Arrow Keys/WASD: Move character
- ESC: Return to menu/pause
